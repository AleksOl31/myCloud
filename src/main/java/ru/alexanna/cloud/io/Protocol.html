Команды клиента

1. Запрос с сервера списка текущей директории

1) Отправка типа команды (write(byte)) - Command.GET_LIST = 31

2. Upload file

1) Отправка типа команды (write(byte)) - Command.POST_FILE = 32

2) Отправка имени файла (writeUTF)

3) Отправка размера файла (writeLong(size)

4) Отправка файла байтовыми массивами (write(byte[] b, int off, int len) по 8192 байт

3. Download file

1) Отправка типа команды (write(byte)) - Command.GET_FILE = 33

4. Запрос на изменение текущей директории

1) Отправка типа команды (write(byte)) - Command.CHANGE_PATH_REQUEST = 34
