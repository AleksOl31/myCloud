Команды клиента

1. Запрос с сервера списка текущей директории

1) Отправка типа команды (byte) - Command.GET_LIST = 31

2. Upload file

1) Отправка типа команды (byte) - Command.POST_FILE = 32

2) Отправка имени файла (String)

3) Отправка размера файла (long)

4) Отправка файла байтовыми массивами (write(byte[] b, int off, int len) по 8192 байт

3. Download file

1) Отправка типа команды (byte) - Command.GET_FILE = 33

4. Запрос на изменение текущей директории

1) Отправка типа команды (byte) - Command.CHANGE_PATH_REQUEST = 34

5. Аутентификация

1) Отправка типа команды (byte) - Command.AUTH = 10

2) Отправка имени пользователя и пароля (String + \\s + String)

3) Сервер отправляет ответ (byte) - Command.AUTH_OK = 10 или GET_BAD_CRED = 13
